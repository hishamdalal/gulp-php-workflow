# Gulp PHP Workflow

### About

Rapid web development with php, scss and javascript.

### Requirments

- node: v20.18.0
- npm: 10.9.0
- gulp-cli: 3.0.0
- gulp: 5.0.0

### How to use:

#### Clone repo

```cmd
git clone https://github.com/hishamdalal/gulp-php-workflow.git
```

#### Change directory

```cmd
cd gulp-php-workflow
```

#### install packages

```cmd
npm install
```

#### Start development

```cmd
npm start
```

#### Build

```cmd
npm run build
```
